# Node REST Vapors
Node REST API boilerplate

## Tools 
1. Node Js (express)
2. Typescript
3. RestServices
4. TypeOrm
5. Prettier
6. Eslint

## Get Started

1.) Clone repository `https://github.com/salman-codewar/node-rest-vapors.git`

2.) `yarn install` in directory

3.) `yarn dev` to spin up server in development
